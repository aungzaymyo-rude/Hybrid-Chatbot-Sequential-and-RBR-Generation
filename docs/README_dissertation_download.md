# Dissertation Draft Download

If binary downloads are blocked in your Codex UI, use the text version:

- `docs/Hybrid_Chatbot_Dissertation_Draft.md`

Binary copies are also present at:
- `Hybrid_Chatbot_Dissertation_Draft.docx`
- `docs/Hybrid_Chatbot_Dissertation_Draft.docx`

## Download options

### Option 1: From GitHub UI
1. Open `docs/Hybrid_Chatbot_Dissertation_Draft.docx`.
2. Click **Download** (or **Raw** then save).

### Option 2: From terminal
```bash
git pull
cp docs/Hybrid_Chatbot_Dissertation_Draft.docx ~/Downloads/
```

### Option 3: Zip only this file
```bash
zip dissertation_draft.zip docs/Hybrid_Chatbot_Dissertation_Draft.docx
```

If you want, this can be renamed to your student/project title before submission.
